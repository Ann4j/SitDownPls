# PUG, SCSS, Webpack

## Запуск

Установить необходимы зависимости:

```shell
    npm install
```

Выполнить команду:

```shell
    npm run serve
```

## Сборка

```shell
    npm run build
```

#### Структура

В проекте присутсвует 3 реализация 3х страниц: главной, каталог и карточка товара "Диван тканевый “D-31”".
